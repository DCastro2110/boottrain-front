# Onboarding Page

## Requirements

- If the user not have weight, height, age or body fat percentage, always that user log in should redirect to ai page to fill that information, and only after that, the user can access to the rest of the app.
- The page should be equal to the Node ID: OY1JW
- Use the components of @src/components/ai
- Use um proxy.ts to validate if user have the required information, if not, redirect to ai page
- The page should be responsive and work on mobile and desktop.
- The name of the page should be onboarding
