# Contexto
A funcionalidade de Workout Plan usa URLs em português, mas atualmente existe uma rota em inglês `src/app/workout-plan/[id]/workout-day/[workoutDayId]/page.tsx` que precisa ser movida para o novo formato.

# Objetivo
Renomear os diretórios para seguir a nova estrutura de rotas em português: `/planos/[planId]/dias/[dayId]`.

# Arquivos Afetados
- `src/app/workout-plan/` (diretório a ser renomeado/removido)
- `src/app/planos/[planId]/dias/[dayId]/page.tsx` (novo destino)

# Instruções de Implementação
1. Crie a nova estrutura de diretórios: `src/app/planos/[planId]/dias/[dayId]/`.
2. Mova o arquivo `page.tsx` (e quaisquer outros arquivos associados presentes) de `src/app/workout-plan/[id]/workout-day/[workoutDayId]/` para a nova pasta `src/app/planos/[planId]/dias/[dayId]/`.
3. Abra o arquivo migrado e atualize quaisquer imports relativos que possam ter quebrado com a movimentação (verifique referências aos componentes, hooks ou à API).
4. Após migrar os arquivos, exclua a pasta antiga `src/app/workout-plan` para manter o projeto limpo e seguir o padrão novo.

# Critérios de Aceite
- A pasta `src/app/workout-plan` não deve mais existir no repositório.
- O arquivo que detalha o dia do treino deve estar localizado corretamente no caminho `src/app/planos/[planId]/dias/[dayId]/page.tsx`.
- A página do dia do treino continua renderizando e compilando corretamente.
