# 1. Setup linting, formatting, and cleanup default files

## Description

Install and configure `prettier`, `eslint-config-prettier`, and `eslint-plugin-simple-import-sort`. Update the Next.js `page.tsx` and `globals.css` files to remove the default Vercel template.

## Acceptance Criteria

- `prettier` and `eslint-config-prettier` are installed and configured.
- `eslint-plugin-simple-import-sort` is installed and configured in `.eslintrc.json` or `eslint.config.mjs`.
- Imports are automatically sorted when formatting/linting.
- `src/app/page.tsx` contains a simple functional component (e.g., `<main><h1>Home</h1></main>`).
- `src/app/globals.css` only contains Tailwind directives and basic resets, removing Next.js default styles.
