# 2. Configure Better Auth client

## Description
Install the `better-auth` package and configure the client-side instance. Also, update Next.js server configuration to treat `better-auth` as an external package.

## Acceptance Criteria
- `better-auth` dependency is present.
- `src/lib/auth-client.ts` is created and exports `authClient` using `createAuthClient` from `better-auth/react`.
- `next.config.mjs` (or `.js`) is updated to include `better-auth` in `serverExternalPackages`.
