# 3. Initialize shadcn/ui

## Description
Run the shadcn/ui initialization to setup the design system configuration, Tailwind variables, and standard utilities.

## Acceptance Criteria
- `npx shadcn@latest init` is run with default/Next.js settings.
- `components.json` is generated.
- `src/lib/utils.ts` (or similar utility file) is present for the `cn` function.
- Tailwind configuration incorporates shadcn CSS variables and plugin.
