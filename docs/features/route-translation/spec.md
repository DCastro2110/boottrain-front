# Feature: Tradução de Rotas

## Objetivo
Traduzir todos os caminhos de URL (rotas) da aplicação Next.js de inglês para português para melhor experiência do usuário local.

## Rotas Mapeadas
- `/` -> `/` (sem alteração de URL)
- `/login` -> `/entrar`
- `/onboarding` -> `/primeiros-passos`
- `/stats` -> `/estatisticas`
- `/workout-plan` -> `/plano-de-treino`
- `/workout-plan/[id]` -> `/plano-de-treino/[id]`
- `/perfil` -> `/perfil` (já em português)

## Impactos
- Navegação (Navbar)
- Redirecionamentos de autenticação
- Links internos
