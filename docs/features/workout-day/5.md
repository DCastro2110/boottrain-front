# Task 5: Finish Session Action and Success Popup

## Description

Implement the functionality to mark the workout session as complete and show a celebratory popup.

## Acceptance Criteria

- [ ] Implement the "Marcar como concluído" button based on design node `gYCIU`.
- [ ] Upon clicking, call `patchWorkoutPlanWorkoutPlanIdWorkoutDaysWorkoutDayIdSessionsSessionId` to mark the session as complete (using the `sessionId` obtained from the start session mutation or existing data).
- [ ] Create a success popup/modal containing the message "Parabéns! Você concluiu o treino do dia!".
- [ ] Include a "Voltar para o menu" button in the popup that, when clicked, closes the popup (or redirects if that makes more sense contextually, but prompt specifies "para fechar o popup").
- [ ] Ensure the popup is accessible and matches the project's styling.
