# Task 7: Test the complete onboarding flow

## Description

Verify the entire flow works end-to-end.

## Requirements

- New user logs in → redirected to /onboarding
- User completes onboarding via AI chat
- User is redirected to home page
- Subsequent logins go directly to home (no re-onboarding)

## Acceptance Criteria

- User without profile fields cannot access home page directly
- User is redirected to /onboarding immediately after login
- AI conversation successfully collects all four fields
- Profile update API call succeeds
- User is redirected to home after completing onboarding
- Returning user with complete profile goes straight to home

## Notes

- Test with a user account that has null profile fields
- Verify the redirect happens both from login and home
- Check that AI conversation flow collects all required data
- Ensure profile update persists correctly
- Test on both mobile viewport and desktop
