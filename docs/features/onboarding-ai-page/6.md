# Task 6: Add responsive styling

## Description

Ensure the onboarding page works on desktop as well as mobile.

## Requirements

- Mobile: full-screen chat experience
- Desktop: centered modal/container experience
- Consistent styling with design system
- Use Tailwind CSS classes for responsiveness

## Acceptance Criteria

- On mobile (< 640px): full-screen chat
- On desktop (>= 640px): centered container with max-width
- Consistent spacing, typography, and colors with design system
- No horizontal overflow on any screen size

## Notes

- Design shows 393px width on mobile (safe area)
- On desktop, can have max-width of 393px and center
- Use `h-[85vh]` for modal height (85% of viewport height)
- Consider using `sm:` breakpoint for desktop styles
