# Task 5: Connect AI chat to collect user data

## Description

Wire up the AI modal to collect weight, height, age, body fat percentage through conversation.

## Requirements

- AI chat interface functional with send/receive messages
- AI is instructed to collect the four required fields
- After collection, call `putUsersUserId` to save data
- Redirect to home after successful data save

## Acceptance Criteria

- User can send messages and receive AI responses
- AI collects: weight, height, age, body fat percentage
- Upon collecting all fields, API call updates user profile
- After successful update, user is redirected to home page
- If profile already complete, user goes directly to home

## Notes

- The AI backend needs proper system prompt to guide conversation
- Use `initialMessage` prop on AIModal to start the conversation
- Consider adding a loading state while saving profile data
- Handle errors gracefully with user-friendly messages
