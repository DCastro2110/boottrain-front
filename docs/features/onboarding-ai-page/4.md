# Task 4: Build onboarding page UI matching design OY1JW

## Description

Create the onboarding page with AI chat interface matching the design spec from Node ID `OY1JW` in `docs/design.pen`.

## Requirements

- Header with "Coach AI" title and Online indicator (green dot)
- Welcome messages in gray chat bubbles
- "Vamos configurar seu perfil?" message displayed
- "Começar" action button
- Chat input at bottom with placeholder "Digite sua mensagem"
- Mobile-first design (393px width), responsive to desktop
- Use existing AI components from `@/components/ai`

## Acceptance Criteria

- UI matches design OY1JW closely
- Header shows Coach AI branding with online status
- Welcome message sequence displayed correctly
- "Começar" button triggers AI conversation
- Chat input is functional at bottom of screen
- Responsive layout works on mobile and desktop

## Notes

- Design uses Inter Tight font, #2b54ff primary color
- Chat bubbles have gray background (#f1f1f1)
- Corner radius 12px for message containers
- The page should feel like a full-screen chat experience
