# Task 6: Implement Home page (main page.tsx)

## Description

Implement the main Home page as a server component that fetches data using `getHomeInfo` from the orval API and passes the data to child components via props. This is the main integration task that ties everything together.

## Files to Create/Modify

- `src/app/page.tsx` (create/replace)

## Implementation Details

1. Create a server component (default for Next.js App Router page - no `"use client"` directive)
2. Fetch data using `getHomeInfo` from `@/lib/api/booTrainAPI` directly (server-side)
3. Structure matching design [Node ID: JflQA]:
   - Banner section with user greeting ("Olá, [Name]", "Bora treinar hoje?")
   - ConsistencyBoard section (pass consistency and fireSequence as props)
   - TreinoDeHoje section (pass todayWorkoutDay as prop)
   - Navbar at bottom
4. Banner content:
   - "Fit.ai" logo text
   - User greeting (from API or session)
   - Motivational text "Bora treinar hoje?"
   - Account badge
5. Pass fetched data to child components as read-only props
6. Use proper section structure with titles ("Consistência", "Treino de Hoje")
7. Page should be responsive and adapt to different screen sizes
