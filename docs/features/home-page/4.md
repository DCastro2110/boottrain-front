# Task 4: Create Navbar component

## Description

Create a `Navbar` server component with bottom navigation containing 5 interactive icons that link to corresponding pages.

## Files to Create

- `src/components/Navbar.tsx`

## Implementation Details

1. Create a server component (no `"use client"` directive)
2. Use `next/link` for navigation
3. Icons and their target pages:
   - House icon → `/` (Home)
   - Calendar icon → `/plano` (Workout Plan)
   - Sparkles icon (filled blue) → `/ai` (AI)
   - Chart icon → `/evolucao` (Stats)
   - User icon → `/perfil` (Profile)
4. Use `lucide-react` for all icons
5. Styling matching design [Node ID: tXHdU]:
   - Fixed at bottom, full width
   - White background with subtle border
   - Corner radius: 20px top corners only
   - 5 icons evenly spaced with 24px gap
   - Each icon in a touch-friendly container (24px icon, 12px padding)
6. The AI (sparkles) icon should be highlighted with blue fill

## Acceptance Criteria

- [ ] Component is a server component at `src/components/Navbar.tsx`
- [ ] Uses next/link for client-side navigation
- [ ] Has 5 icons: house, calendar, sparkles, chart, user
- [ ] Sparkles icon has blue fill/highlight
- [ ] All icons link to correct pages:
  - House → /
  - Calendar → /plano
  - Sparkles → /ai
  - Chart → /evolucao
  - User → /perfil
- [ ] Styled with white background, subtle border, 20px top radius
- [ ] Icons are evenly spaced with proper touch targets
- [ ] Component can be imported from `@/components/Navbar`

## Dependencies

- lucide-react for icons (House, Calendar, Sparkles, BarChart3, User)
