# Task 3: Create RestDayCard component

## Description

Create a `RestDayCard` client component that displays a rest day UI when the user has no scheduled workout for today. This component receives data via props from the parent (server) page component.

## Files to Create

- `src/components/RestDayCard.tsx`

## Implementation Details

1. Create a client component (`"use client"` directive)
2. Props interface:
   ```typescript
   interface IRestDayCardProps {
     weekday: string;
   }
   ```
3. Display content matching the design [Node ID: hOE4O]:
   - Weekday badge at top (e.g., "quarta") with subtle dark background
   - Zap icon in blue
   - "Descanso" text in large text (24px)
4. Light gray background (`bg-neutral-100` equivalent)
5. 12px corner radius, 110px height

## Acceptance Criteria

- [ ] Component is a client component at `src/components/RestDayCard.tsx`
- [ ] Props interface accepts weekday string
- [ ] Displays weekday name in a badge with subtle dark background
- [ ] Displays zap icon in blue color
- [ ] Displays "Descanso" text in large text (24px, weight 600)
