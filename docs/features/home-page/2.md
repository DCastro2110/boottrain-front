# Task 2: Create WorkoutCard component

## Description

Create a `WorkoutCard` client component that displays today's workout information including the workout name, weekday name, estimated duration, and number of exercises. This component receives data via props from the parent (server) page component.

## Files to Create

- `src/components/WorkoutCard.tsx`

## Implementation Details

1. Create a client component (`"use client"` directive)
2. Props interface:
   ```typescript
   interface IWorkoutCardProps {
     workout: {
       date: string;
       name: string;
       estimatedDurationInSeconds: number;
       numberOfExercises: number;
       coverImageUrl: string | null;
       isCompleted: boolean;
     };
   }
   ```
3. Display content matching the design [Node ID: YNMOw]:
   - Weekday badge at top (e.g., "SEXTA") with blur effect background
   - Workout name (e.g., "Superiores") in large text (24px)
   - Estimated duration with timer icon (e.g., "45min")
   - Number of exercises with dumbbell icon (e.g., "4 exercícios")
4. Use background image if `coverImageUrl` is provided, otherwise use gradient overlay
5. Dark overlay on background image (`bg-black/40` equivalent)

## Acceptance Criteria

- [ ] Component is a client component at `src/components/WorkoutCard.tsx`
- [ ] Props interface is properly typed with all workout fields
- [ ] Displays weekday name in a badge with blur background
- [ ] Displays workout name in large text (24px, weight 600)
- [ ] Displays estimated duration with timer icon
- [ ] Displays number of exercises with dumbbell icon
- [ ] Shows background image if coverImageUrl is provided
- [ ] Shows dark gradient overlay for text readability
- [ ] Component can be imported from `@/components/WorkoutCard`

## Dependencies

- lucide-react for icons (Timer, Dumbbell)
