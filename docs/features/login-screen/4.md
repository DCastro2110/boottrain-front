# Task 4: Add Login Page Styling with Tailwind

## Objective

Apply styling to the login page to match the Pencil design (Node ID: Mv75i).

## Deliverables

- Updated `src/app/login/page.tsx` with Tailwind styling
- Styling matches the 402x801 Login frame design

## Requirements

### Background Styling

- Full viewport height (`min-h-screen`)
- Dark background with image (use placeholder or brand gradient)
- Dark overlay: `bg-black/70` or similar

### Card Styling

- Blue gradient background: `bg-gradient-to-b from-blue-500 to-blue-600` or exact brand color `#2b54ff`
- Top rounded corners: `rounded-t-2xl` or `rounded-t-xl`
- Padding: `px-12 py-5` (48px horizontal, 20px vertical)
- Gap between elements: `gap-16` or similar
- Center content vertically and horizontally

### Typography

| Element     | Style                                                      |
| ----------- | ---------------------------------------------------------- |
| Hero text   | Inter Tight, 32px, font-semibold (600), white, text-center |
| Button text | Inter, 14px, font-semibold (600), black                    |
| Copyright   | Inter Tight, 12px, text-white/70                           |

### Logo

- Fit.ai text logo at top
- White color
- Centered horizontally

### Layout Structure

```
Full viewport container (dark bg + overlay)
  └── Centered content
        └── Fit.ai logo (top)
        └── Blue card
              ├── Hero message
              ├── Google button
              └── Copyright footer
```

### Acceptance Criteria

- [ ] Background matches design (dark with overlay)
- [ ] Blue card visible with correct rounding
- [ ] Hero text styled correctly (32px, white, centered)
- [ ] Google button matches pill shape design
- [ ] Copyright footer visible with correct opacity
- [ ] Logo visible at top
- [ ] Responsive on mobile and desktop
